Java는 반복문에 이름을 붙일 수가 있단다. 반복문을 원하는 조건에 따라 제어 하고 싶으면 `break`,`continue`문을 사용하는데 이 구문은 기본적으로 자신이 속한 바로 상위의 반복문만 멈추게 할 수 있다. 그런데 반복문에 이름을 붙이면, 중첩 반복문일 경우 원하는 특정 반복문을 중단 시킬 수 있다는 것이다.

이름있는 반복문을 사용하여 원하는 계산 방식으로 값을 계산 해주는 프로그램을 만들어보자.

1. 먼저 원하는 계산 방식과 계산하고 싶은 숫자 대상을 저장해줄 변수를 만들어준다.
2. 화면 입력을 위한 `Scanner` 객체를 생성한다.

```java
import java.util.Scanner;

public class NamedLoop {
  public static void main(String[] args){
    int menu = 0;
    int num = 0;
    Scanner scanner = new Scanner(System.in);
  }
}
```

3. while문에 `outer`라는 이름을 붙여주고 입력받은 메뉴를 저장해준다.

```java
import java.util.Scanner;

public class NamedLoop {
    public static void main(String[] args){
    //...
     outer: //while문의 이름
     while(true) { // 무한 반복
        System.out.println("(1) square");
        System.out.println("(2) square root");
        System.out.println("(3) log");
        System.out.print("원하는 메뉴(1~3)를 선택하세요. (종료:0)>");

        String tmp = scanner.nextLine(); // 입력받은 내용 저장
        menu = Integer.parseInt(tmp);
   }
}

}
```

4. 조건문에 따라 프로그램 종료여부를 결정한다.

```java
import java.util.Scanner;

public class NamedLoop {
  public static void main(String[] args){
    //...
     if(menu == 0) {
        System.out.println("프로그램을 종료합니다.");
        break;
      } else if ((1 < menu && menu > 3)) {
        System.out.println("메뉴를 잘못 선택하셨습니다.(종료는 0)");
        continue;
      }
  }


}
```

5. 무한 for문 안에서 조건에 따라 `outer`(while문)을 종료 여부를 결정한다.

```java
import java.util.Scanner;

public class NamedLoop {
  public static void main(String[] args){
    //...
    for(;;) {
        System.out.println("계산할 값을 입력하세요 (계산 종료: 0, 전체 종료: 99)");
        tmp = scanner.nextLine();
        num = Integer.parseInt(tmp);

        if(num == 0) { // 계산 종료
          break;
        }

        if(num == 99) { // 전체종료
          break outer;
        }

        switch (menu) {
          case 1:
            System.out.println("result=" + num*num);
            break;
          case 2:
            System.out.println("result="+ Math.sqrt(num));
            break;
          case 3:
            System.out.println("result="+Math.log(num));
            break;
        }
    }
  }

}
```
