## 변수의 타입

### 레퍼런스형 (Reference Type)

레퍼런스 형은 객체 연결을 위한 타입으로 사용된다. 아래 예시코드에서느 변수 앞에 붙은 `Marine`은 레퍼런스 타입이고 해당 변수는 레퍼런스 변수가 된다.

```java
Marine marine = new Marine("지윤", 80);
```

### ✅ 원시형 변수

원시형 변수에는 값 자체가 담기지만 레퍼런스 형은 데이터의 주소값을 가르킨다. 따라서 레퍼런스형의 변수를 사용할 때는 해당 변수에 담긴 주소값을 찾아가서 데이터에 접근하게 되기 때문에, 참조형이라고 하는 것이다.

타입을 지정할 때도 참조형은 대문자로 해당 객체를 생성하는 클래스를 타입으로 가지고, 원시형 변수는 소문자로 시작한다.

이런식으로 객체를 참조형으로 저장하는 이유는 객체의 크기가 유동적이기 때문에, 하나의 변수에 담기에는 너무 크기 때문이다.
