## 🛠 배열 만들기

### 1. 원소 넣어서 만들어주기

배열을 만드는 가장 간단하고 기본적인 방법은 아래처럼 직접 배열의 원소를 넣어서 만들어주는 것이다.

```javascript
const fruit = ['사과', '바나나']
```

### 2. Array.from()

`Array.from()`메서드를 사용하면 유사배열 객체나 반복 가능한 객체(iterable object)를 얕게 복사해 새로운 객체를 만들수 있다.

```javascript
Array.from(
  '배열로 변환하고자 하는 객체 - arrayLike',
  '배열의 모든 요소에 대해 호출할 매핑 함수 - mapFn',
  'mapFn 실행시 this로 사용할 값'
)
```

```javascript
const foo = Array.from('foo') // ['f','o','o']
const number = Array.from([1, 2, 3]) // [1,2,3]
const number2 = Array.from([1, 2, 3], (num) => num + num) // [2,4,6]
const number3 = Array.from({ length: 5 }, (v, i) => i) // [0,1,2,3,4]
```

위 예시에서 마지막 `number3`를 살펴보면 두번째 콜백함수의 인자가 두개인데 `v`는 배열의 원소값이고 `i`는 인덱스 값이다. 그리고 콜백함수의 리턴되는 값을 실제 원소가 되어 새로운 배열을 만들어준다.
위 예시에서는 정해진 배열 없이 배열의 길이만 지정해주었기 때문에, 실질적으로는 `'undefined'만 있는 길이가 5인 배열에서 새로운 배열을 만들어줘`, 라는 뜻이 된다. 따라서 위 예시에서 `v`는 `undefined`가 된다.

#### 🛠 Array.from() 으로 시퀀스 생성기 만들어주기

Array.from() 메소드를 활용하면 원하는 범위 내에서 특정 조건의 원소값을 갖는 배열을 추출할 수 있다.
아래 예시에서 `range` 함수는 원하는 길이만큼 원하는 숫자의 거듭제곱을 배열로 담게 된다.

```javascript
const range = (start, stop, step) =>
  Array.from({ length: stop - start }, (_, i) => Math.pow(step, i + 1))

range(1, 10, 3)
//[(3, 9, 27, 81, 243, 729, 2187, 6561, 19683)]
```

### 3. fill() 사용하기

`fill` 메소드를 사용하면 지정된 길이만큼 동일한 원소로 채워서 만들어줄 수 있다.

```javascript
const one = new Array(3).fill(1)

// [1,1,1]
```
