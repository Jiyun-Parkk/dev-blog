Node.js는 크롬 자바스크립트 엔진으로 빌드된 자바스크립트 `런타임 환경`이다. 본래 Javascript는 웹 브라우저에서만 동작하는 언어였다. 웹 브라우저에 내장된 자바스크립트 엔진으로 실행되는 코드이기 때문에 웹에서만 실행 할 수 있었다. 하지만 Node.js가 등장하면서 웹 브라우저 환경이 아니어도 Javascript를 실행 할 수 있게 되었다. 다시말해서 웹브라우저로부터의 종속성에서 벗어난 것이다.

Node.js를 사용하려면 공식홈에서 Node.js를 다운 받아야 한다. 공식 홈에는 두가지 버전이 있는데 LTS 버전이 범용적으로 사용되므로 LTS버전을 다운 받는 것을 추천한다.

Node.js를 다운 받고 아래 명령어로 버전을 확인 해볼 수 있다.

```bash
node -v
```

Node.js가 설치되면 이제 VS Code 터미널에서 자바스크립트를 실행 해볼 수 있다. test로 javascript 파일을 만들고, `node`명령어로 해당 파일을 실행시키면 터미널에 콘솔이 찍히게 된다.

```javascript
// test.js
console.log('test')
```

```bash
node test.js
```

결국 node.js는 간략히 말하면 Javascript의 실행기라고 보면 된다. 이렇게 Javascript가 웹브라우저의 종속성에서 벗어난 것은 큰 의미가 있다. Javascript로 서버 코드를 작성하면 node.js가 실행시키는 것이 가능해지기 때문에 백엔드의 영역까지 넘어설 수 있게 되었기 때문이다.

기존의 Javascript는 프론트개발에서만 의미가 있었는데 이제는 백엔드영역까지 넘어가면서, 웹페이지 뿐만 아니라 웹앱, 게임 등 다양한 분야에서 개발이 가능해졌다. 최근에는 Javascript로 AI 개발까지 성공했다는 사례가 있어서, Javascript 하나만 잘 공부해도 스펙트럼이 엄청 넓어지게 되는 것이다.
